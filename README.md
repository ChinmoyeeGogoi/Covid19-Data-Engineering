# COVID-19 ETL Pipeline

This project demonstrates an end-to-end ETL pipeline that extracts the latest COVID-19 data from the Our World in Data (OWID) API, transforms and cleans the dataset using Python and Pandas, and loads it into an Azure SQL Database. The pipeline is designed to be orchestrated using Azure Data Factory (ADF), showcasing cloud-based data ingestion, transformation, and storage practices commonly used in modern data engineering workflows.
